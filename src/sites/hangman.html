Hangman a word guessing game

Hangman game

The object of the game is to guess the secret searched word by guessing letters.
Each incorrect guess brings you closer to being "hanged." This game helps to sharpen spelling and word-decoding skills.

The searched word will be randomly generated by the program and displayed with spaces (_)
If the searched word is house, it will be displayed as _ _ _ _ _

Every time you correctly guess a letter which is included in the searched word, the equivalent space will be replaced by the letter
You enter 's' it will display _ _ _ s _

This project is build in Java
