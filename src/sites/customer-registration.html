Online Shop

Customer Registraion

This project is about a prototype online Shop
It consists out of the 5 main modules "Shop", "Ordering Process", "User Account", "Product-Manager-Interface", "Admin Area" and covers all typical functionalities of a real online shop.

In the Shop the customer can search for products view them, add them to the shopping cart and eventually buy them

A customer can also sign up as user which has access to his personal account where he can modify his data, manage previous orders, save his shopping cart and contact the shop support via chat

The Product-Manager-Interface is used by Product Managers to manage users, create and update product data, support users via chat

The admin area is used by administrators to administrate users, manage product managers and perform settings

Java, SpringBoot, MySQL, HTML, JavaScript, CSS, Maven, Jenkins, Docker, Git were mainly used to build this Online Shop Web Application
